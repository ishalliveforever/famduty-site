Privacy Policy

Privacy information for FamDuty

FamDuty is built to help families manage chores, rewards, and routines. This page describes how the current app implementation handles information as of March 31, 2026.

Overview

FamDuty is designed to help families manage chores, rewards, and household routines.

FamDuty does not use a separate FamDuty account system in the current app implementation. Information created in the app is stored on the device and may also be synced through Apple's CloudKit service using the app's configured iCloud container.

Information FamDuty stores

FamDuty stores information entered or created in the app so the app can function. This may include family setup details, parent and child names, child invite contact information, chores, chore templates, rewards, reward requests, family announcements, profile customization, activity history, and related household app state.

On-device storage

FamDuty stores information locally on the device using iOS local storage. The current implementation also uses Keychain-backed storage for certain persisted device and profile access information.

Cloud syncing

When cloud sync features are used, FamDuty syncs family data through Apple's CloudKit service using the app's configured iCloud container. In the current implementation, CloudKit is used for family snapshots, invites, chore submissions, chore review decisions, reward requests, chore assignments, reminders, and child profile records.

Device features and permissions

FamDuty may use notifications, Face ID, camera access for photo proof, and on-device storage for profile photos and related images.

How FamDuty uses information

FamDuty uses stored information to provide family setup, parent and child access, chore management, approvals, rewards, activity history, notifications, and syncing across participating devices.

Data sharing

FamDuty is intended for use within a household. This website does not describe selling personal information. The current app implementation reviewed for this site uses local device storage and Apple's CloudKit and iCloud services for its core data flow.

Data control

Families can reset or remove family information from within the app. If FamDuty's storage model, connected services, or account structure change in the future, this privacy policy should be updated to match those changes.

Children's information

FamDuty is built for family use under parent oversight. Parents are expected to manage setup, approvals, invitations, and household participation for child profiles.

Contact

If you have privacy questions, contact famdutyapp@gmail.com.

Policy updates

This privacy policy may be updated as FamDuty evolves. The effective date should be updated whenever material changes are made.
