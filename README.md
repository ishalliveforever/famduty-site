# FamDuty GitHub Pages Site

Static landing site for FamDuty, designed for easy GitHub Pages publishing.

## Files

- `index.html` - main landing page
- `support/index.html` - support page
- `privacy/index.html` - privacy policy page
- `styles.css` - site styling
- `script.js` - small scroll reveal enhancement
- `assets/famduty-app-icon.png` - app icon reused from the iOS app

## Publish with GitHub Pages

### Option 1: Publish this folder from a repository root

1. Put the contents of this folder in the root of a GitHub repository.
2. Push to GitHub.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your branch, usually `main`.
6. Select `/ (root)` and save.

### Option 2: Publish this folder as `/docs`

1. Copy this folder into your repository as `docs/`.
2. Push to GitHub.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your branch and choose `/docs`.
6. Save.

## Before launch

- Replace `support@famduty.app` if you want a different support email.
- Replace the App Store CTA link when your App Store URL is ready. The TestFlight link is already wired in.
- Swap the screenshot placeholders with real screenshots if you want.
